2.1 Algorithms, Selection, and Iteration

  • Algorithm → Step-by-step process to solve a problem.
  • Selection → Choosing between paths using if, if-else.
  • Iteration → Repeating steps using loops (for, while).

2.2 Designing Algorithms with Selection and Iteration

if(score >= 60) {
    System.out.println("Pass");
} else {
    System.out.println("Fail");
}

for(int i = 1; i <= 5; i++) {
    System.out.println(i);
}

2.3 Using the Accumulator Pattern

  • Initialize accumulator variable before loop.
  • Update accumulator inside loop.
  • Use value after loop.
int sum = 0;
for(int i = 1; i <= 100; i++) {
    sum += i;
}
double average = (double)sum / 100;

2.4 Input-Controlled Loops

Scanner scan = new Scanner(System.in);
int number = scan.nextInt();
int sum = 0, count = 0;
while(number != -1) {
    sum += number;
    count++;
    number = scan.nextInt();
}
double average = (double)sum / count;

2.5 Loop with if: Minimum and Maximum

int min = Integer.MAX_VALUE;
int max = Integer.MIN_VALUE;
for(int i = 0; i < n; i++) {
    int num = (int)(Math.random() * 100);
    if(num < min) min = num;
    if(num > max) max = num;
}

2.7 Finding Digits

while(number > 0) {
    int lastDigit = number % 10;
    if(lastDigit == digit) return true;
    number /= 10;
}
return false;

2.8 Frequency

int count = 0;
for(int i = 1; i <= 100; i++) {

}

2.10 Implementing String Algorithms

// 2.10.1 Find & Replace
int index;
while(message.indexOf("1") >= 0) {
    index = message.indexOf("1");
    message = message.substring(0,index) + "l" + message.substring(index + 1);
}

// 2.10.2 For loop traversal
for(int i = 0; i < s.length(); i++) {
    String ch = s.substring(i,i+1);
}

// 2.10.3 Count characters
int count = 0;
for(int i = 0; i < s.length(); i++) {
    if(s.substring(i,i+1).equals("e")) count++;
}

// 2.10.4 Reverse string
String sReversed = "";
for(int i = 0; i < s.length(); i++) {
    sReversed = s.substring(i,i+1) + sReversed;
}

// 2.10.5 Replace "cat" with "dog"
while(message.indexOf("cat") >= 0) {
    index = message.indexOf("cat");
    message = message.substring(0,index) + "dog" + message.substring(index + 3);
