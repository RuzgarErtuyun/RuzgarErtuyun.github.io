BIG-IDEAS

AP CSA Unit 1

Algorithm

An algorithm is a step by step process to follow when completing a task or solving a problem.

It is helpful to be very specific and detailed when writing algorithms.

Data Types

There are many data types like integers, doubles, and booleans.

Integers are negative and positive whole numbers, doubles are real numbers, and booleans are true or false.

Printing

Use System.out.print for printing in Java.

Comments

Use // to add a comment in your code.

Documentation

Documentation is important so that code is made clearer.

An instruction manual is a good example of documentation.

An API is like an instruction manual.

Java Keywords

  • public → Can be accessed from anywhere in the program.
  • class → A blueprint for objects in Java; all code must be inside a class.
  • static → Belongs to the class itself, not to any object.
  • void → A method that doesn’t return a value.
  • main → The starting point of a Java program.

Skill #2 – API

API (Application Programming Interface): Documentation that explains what a library can do and how to use its methods—like an instruction manual for code.

1.5–1.9 (Incrementing a Variable)

a = a + 1;
